Personalized-Event-tracker:

The idea is to create an agent that gives a user a daily update for activities and events the user might be interested with personalized recommendations

- The user creates a profile chose a city (in the first project stage only Hamburg is selectable) and selects a few interests
- Events from the city a fetched on a daily basis from similar event pages without user personalization
- The user gets recommendations and selects (i am interested / i am not interested)
- Via the user feedback the app improves their future recommendations
- The user has a calendar to see his the events he/she is interested into
- There are options to activate email reminder for specific events.
